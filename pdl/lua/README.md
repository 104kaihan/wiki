Lua
===

[Lua 官方網站](http://www.lua.org/about.html)所提到的特色：

* Lua 既小且快又不用錢。
* Lua 是非常易學易用的的一種語言。
* Lua 與 C/C++ 的整合性很好。
* Lua 的可移植性極高。
* Lua 是用 ANSI C 寫的
