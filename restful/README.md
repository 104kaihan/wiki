# RESTful

## HTTP Method

| Method | Description |
| ------ | ----------- |
| GET | 讀取資源 (safe & idempotent)|
| PUT | 替換資源 (idempotent) |
| DELETE | 刪除資源 (idempotent) |
| POST | 新增資源；也作為萬用動詞，處理其它要求 |
| PATCH | 更新資源部份內容 |
| HEAD | 類似GET，但只回傳HTTP header (safe & idempotent) |

> 可參考 [wiki](https://en.wikipedia.org/wiki/Hypertext_Transfer_Protocol#Request_methods)
