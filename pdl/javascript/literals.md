Literals
========

JavaScript 提供許多實字定義方式，讓程式寫起來更簡單，也更具表達力。

Object Literals
---------------

在 JavaScript 使用實字定義一個物件非常簡單，語法如下：

```javascript
var foo = {};
```

Array Literals
--------------

Array 也非常簡單，語法如下：

```javascript
var foo = [];
```

Function Literals
-----------------

Function 也可以用實字定義，詳細可以參考[該文件](function.md)
 
 ```javascript
var foo = function() {};
```

RegExp Literals
---------------

RegExp 實字定義如下：

```javascript
var foo = /bar/;
```
