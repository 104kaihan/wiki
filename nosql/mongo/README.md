MongoDB
=======

NoSQL 的一種，它使用檔案導向來管理資料庫

Basic
-----

RDBMS 與 MongoDB 的名詞對照表：

|  RDBMS  |  MongoDB  |
|  -----  |  -------  |
| Table   | Collection |
| Column  | Key |
| Value   | Value |
| Records / Rows | Document / Object |
